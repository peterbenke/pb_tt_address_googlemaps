Installation
============

#. Download extension "tt_address" (min. version 3.1.0) and install it with the extension manager or via composer: https://typo3.org/extensions/repository/view/tt_address

#. Download this extension "pb_tt_address_googlemaps" and install it with the extension manager or via composer

#. Go to extension configuration and input your Google API Key.
   How to get an API key: https://developers.google.com/maps/documentation/javascript/get-api-key

#. Include the static typoscript from this extension in your template
