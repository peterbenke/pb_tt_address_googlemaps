Users manual
============

#. Create your address data (tt_address) in a sysfolder

   #. Input the address

   #. Click on the wizard icon to get the coordinates for this address

   #. Save the coordinates

   #. Save the address


#. Create a plugin "Address list (google maps)"

   #. Activate "Google maps", if wanted

   #. Choose the order of the listed data

   #. Choose output type of microdata (json-ld), for more info see:

      #. https://schema.org/docs/gs.htm
      #. https://schema.org/Person
      #. https://schema.org/LocalBusiness

   #. Choose the folder, where the addresses are stored
